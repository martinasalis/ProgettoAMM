Entita':- Utenti- Post- GruppiEntita' con attributi:  Utenti:- Id- Nome- Cognome- Compleanno- Descrizione- Url immagine- Password Post- Id- Contenuto- Tipo post- Url Gruppi- Id- NomeRelazioni:- Utenti e Gruppi- Utenti e Post- Post e Gruppi Utenti e Gruppi- Un utente partecipa a piu' gruppi. (Molti)- Ad un gruppo partecipano piu' utenti. (Molti)Relazione Molti a Molti. Utenti e Post- Un utente crea piu' post. (Molti)- Un post e' creato da un solo utente. (Uno)Relazione Uno a Molti Post e Gruppi- Un post e' contenuto in un solo gruppo. (Uno)- Un gruppo puo' contenere piu' post. (Molti)Relazione Molti a Uno.
